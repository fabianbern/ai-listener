Anthropic API-nyckel

Appen transkriberar med telefonens inbyggda tal-till-text (gratis). Anthropic-nyckeln används bara för AI-analysen.

Hämta din nyckel på console.anthropic.com → API Keys. Lagras bara lokalt på din telefon.

💡 Utan nyckel transkriberas talet ändå — men AI-analysen visar ett demo-svar.
